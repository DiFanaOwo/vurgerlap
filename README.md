# Nombre del Proyecto

BurgerLab será una aplicación móvil enfocada en facilitar la experiencia de compra de hamburguesas mediante un sistema rápido y moderno. Los usuarios podrán visualizar el menú digital, conocer ingredientes y precios, realizar pedidos desde el celular y solicitar delivery o pedidos para recoger.
La aplicación también permitirá mejorar la organización del restaurante y reducir la congestión en el local, optimizando el tiempo de atención y brindando una experiencia más cómoda para los clientes.
